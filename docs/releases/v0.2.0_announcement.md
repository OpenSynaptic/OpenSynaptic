## 正文（可直接发布）

我们很高兴发布 **OpenSynaptic v0.2.0**。

本次版本聚焦于三件事：

1. **性能优化可落地**：强化多进程并发参数说明，帮助你在真实负载下更稳定地跑出吞吐。
2. **RS Core（Rust 核心）路径更清晰**：从构建、检查到切换，形成完整操作链路。
3. **优化使用示例更实用**：补齐高负载压测、自动调优、后端对比与 CI 强约束示例。

### 重点更新

- 多进程并发能力：`--processes`、`--threads-per-process`、`--batch-size`
- 自动调优能力：`--auto-profile` + `--profile-*` 参数矩阵
- RS Core 工具链：`rscore-build`、`rscore-check`、`core --set rscore --persist`
- 对比验证能力：`plugin-test --suite compare` 可用于 `pycore` / `rscore` 场景评估

### 为什么值得升级到 v0.2.0

- 更快定位适合你机器的并发配置，而不是靠手工反复试错
- 更低切换成本接入 Rust 核心路径，便于逐步验证加速收益
- 更完整的发布与文档入口，团队协作和交接成本更低

### 建议升级后第一步

```powershell
python -u src/main.py rscore-check
python -u src/main.py plugin-test --suite stress --auto-profile --profile-total 20000 --final-runs 1
python -u src/main.py plugin-test --suite compare --total 5000 --runs 1 --warmup 0 --processes 2
```

> 说明：`v0.2.0` 不包含协议线格式的破坏性变更（No breaking wire-format changes）。

完整更新说明请见：`docs/releases/v0.2.0.md`
