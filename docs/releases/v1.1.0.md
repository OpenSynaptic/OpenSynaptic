---

## Breaking Changes

- None.

No protocol wire-format, API contract, or configuration key removals are introduced in this release draft.

---

## Migration Guide

No runtime migration is required.

Recommended repository maintenance steps:

1. Link internal team docs/wiki to `docs/README.md` as the canonical documentation entry.
2. Track future releases by appending entries to `CHANGELOG.md`.
3. For runtime upgrades, verify `Config.json` keys against `docs/CONFIG_SCHEMA.md` before deployment.

## Verification and Test Notes

This release is documentation-only. If you still want a runtime confidence pass after upgrade, run:

```powershell
python -u src/main.py plugin-test --suite component
python -u src/main.py plugin-test --suite stress --workers 8 --total 200
python scripts/concurrency_smoke.py 200 8 6
```

Native path checks (optional):

```powershell
python -u src/main.py native-check
python -u src/main.py native-build
```

## Known Issues

- Some IDE/static analyzers can retain stale parse diagnostics after heavy in-file edits; if runtime checks pass (`py_compile`/tests), reindex or restart the language service.
- Native-only codec/security code paths require local shared library availability; environments without built artifacts may fail at runtime.
